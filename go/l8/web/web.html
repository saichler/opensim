Powerful Features

Everything you need to simulate realistic network environments and test your applications at scale.

🌐

Multi-Protocol Support

SNMP v2c/v3 (MD5/SHA1 auth, DES/AES128 privacy), SSH with VT100 terminal emulation, and HTTPS REST API simulation.

📈

Massive Scalability

Simulate 25,000+ concurrent devices with network namespace isolation, pre-generated metrics, and lock-free atomic indexing.

🎮

GPU Server Simulation

NVIDIA DGX-A100, DGX-H100, and HGX-H200 with per-GPU utilization, VRAM, temperature, power, fan speed, and clock speed metrics via DCGM OIDs.

📊

Dynamic Metrics

Realistic CPU, memory, temperature, and GPU metrics with sine-wave cycling patterns and per-device-category profiles.

🔧

Network Namespace Isolation

Each device runs in its own Linux network namespace with a dedicated TUN interface and IP address for realistic isolation.

🔄

RESTful API & Web UI

Complete REST API with round-robin and category-based device creation, plus a web interface for real-time management.

⚡

28 Device Types

Routers, switches, firewalls, servers, GPU servers, and storage systems from Cisco, Juniper, NVIDIA, Palo Alto, Arista, Fortinet, and more.

🔒

HTTPS Storage APIs

Secure REST API endpoints for Pure Storage, NetApp ONTAP, Dell EMC Unity, and AWS S3 with shared TLS certificates.

Get Started

Follow these simple steps to get OpenSim running in minutes.

1

Clone & Build

Clone the repository and build the simulator.

git clone https://github.com/saichler/opensim.git cd opensim/go/simulator go build -o simulator .
2

Run Simulator

Start the simulator with root privileges for TUN interface creation.

sudo ./simulator
3

Create Devices

Create your first simulated devices using the API or web interface.

curl -X POST http://localhost:8080/api/v1/devices \ -H "Content-Type: application/json" \ -d '{"start_ip":"192.168.100.1","device_count":5,"netmask":"24"}'

Experience the Power

OpenSim provides a comprehensive web interface for managing your simulated network devices. Create, monitor, and interact with thousands of devices through an intuitive dashboard.

  • GPU server simulation with NVIDIA DGX/HGX metrics
  • SNMPv3 support with authentication and privacy
  • Dynamic CPU, memory, temperature, and GPU metrics
  • CSV export, routing scripts, and advanced filtering
  • HTTPS REST APIs for storage device simulation
